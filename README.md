# Library Database Management System

Java-based backend application built using Spring Boot and REST APIs.

## Features
- RESTful APIs for book management
- Java backend using OOP principles
- CRUD operations
- Basic data handling
- API testing via Postman

## Technologies Used
- Java
- Spring Boot
- REST APIs
- SQL (conceptual)
- Postman

## API Endpoints
- GET /api/books
- POST /api/books
